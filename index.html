    How to use?

    1. Create a template for repetitive lines you need to write, mark the wildcards (variables) with the ` (grave accent) character

      For example, copy the following into the Template box:

      <li>Hello there, `first_name`. `greeting`</li>

    2. Add a newline after inserting the above into the Template box (for a cleaner output)

    3. Type a user's name into the Set highlighted value to... box and press Insert value

      Follow this up with a greeting, such as How are you?

    4. Repeat step 3 a few times with different names and greetings

    5. Review the Output box for your generated content
